
# Seek and Share

Seek and Share is a web application that allows you to read, write and share personal stories, life experiences, testimonies, and daily devotions. 


## Features

- Read stories
- Write stories
- Delete stories
- Save Draft stories
- Publish stories


## Tech Stack

- Next JS
- TypeScript
- Tailwind CSS
- medium-editor
- next-auth
- mongoose
- mongodb
